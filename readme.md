# Screen Recorder Pro

A professional, full-screen screen recorder with system audio and microphone support.

---

##  Features

- **Screen Recording** - Capture your entire screen or specific windows
- **System Audio** - Record audio from your screen (YouTube, games, etc.)
- **Microphone Support** - Add voice commentary to your recordings
- **Pause/Resume** - Pause and resume recording without losing data
- **Preview** - Watch your recording before downloading
- **Download** - Save as WebM format with timestamp
- **Reset** - Clear everything and start fresh
- **Keyboard Shortcuts** - Power user shortcuts for efficiency
- **Responsive** - Works on all screen sizes

---

##  Tech Stack

- **HTML5** - Semantic structure
- **CSS3** - Custom properties, Flexbox, animations
- **Vanilla JavaScript** - No external libraries
- **Web APIs**:
  - MediaStream API
  - MediaRecorder API
  - getDisplayMedia()
  - getUserMedia()
